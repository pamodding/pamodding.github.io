Player Editor

Version 1.6

Information:

This Mod Adds:
+ Allows you to change the player's SPEED, SIZE and COOLDOWNS
+ Allows you to toggle INVINCIBILITY and INVISIBILITY in Editor Fullscreen
+ Allows you to toggle INVISIBILITY in Zen Mode
+ Customizable in-game IF YOU HAVE A CONFIGURATION/CONFIG MANAGER

Dependencies (REQUIRED):
Configuration Manager FOR 20.4.4

Config Manager FOR 21.8.1

Downloads



Download in 6 easy steps:

1. Install this version BepInEx modloader for PA 20.4.4 or this version BepInEx modloader for PA 21.8.1
2. Run the game after BepInEx is installed
3. Install Configuration Manager mod for PA 20.4.4 or Install Config Manager for PA 21.8.1
4. Click "Download for (Your prefered version)" above
5. Place the DLL file in ProjectArrhythmia/BepinEx/plugins
6. Enjoy
